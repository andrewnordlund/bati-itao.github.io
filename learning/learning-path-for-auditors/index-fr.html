Parcours d'apprentissage pour les auditeurs

Parcours d'apprentissage de l'auditeur de documents

Il s'agit d'un parcours d'apprentissage destiné à toute personne souhaitant devenir un auditeur d'accessibilité des documents. Ce parcours d'apprentissage comprend des cours sur les bases de l'accessibilité des documents et sur l'audit de l'accessibilité pour différents types de documents.

Parcours d'apprentissage pour les auditeurs web

Il s’agit d’un parcours d’apprentissage destiné à quiconque souhaite devenir vérificateur de l’accessibilité du Web. Ce parcours d’apprentissage comprend des cours sur les principes de base de l’accessibilité, le développement Web et la vérification de l’accessibilité pour les applications Web et mobiles.
